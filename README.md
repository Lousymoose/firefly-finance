# Firefly III Finance Tracker

Self-hosted personal finance manager using Docker.

## Setup

1. Copy `.env.example` to `.env` and configure
2. Copy `.db.env.example` to `.db.env` and configure
3. Ensure database passwords match in both files
4. Run `docker compose up -d`
